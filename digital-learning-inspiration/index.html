如何使用本站

關於本站

《數位教材設計靈感大全》是一套提供教師、教材設計者、教育科技工作者與學生使用的數位教材設計案例資料庫。我們相信程式特效不只是視覺展示，更能轉化為真正有意義的互動教材。

本站收錄 JavaScript 特效、CSS 動畫、自然科互動網站案例，以及 Three.js、GSAP、SVG、Canvas 等教材效果案例，並針對每個案例提供教學應用、技術分析與修改建議。

網站以純 HTML、CSS 與原生 JavaScript 開發，不需後端伺服器、不需登入、不連接資料庫，所有資料儲存於本地端，可直接部署到 GitHub Pages，也可在本機直接開啟 index.html 使用。

數位教材設計靈感大全

將程式特效轉化為有意義的互動教材，供教師、教材設計者與學生自由取用、修改與再創作。

技術說明

以 HTML5、CSS3 與原生 JavaScript 開發；程式碼醒目顯示採用 Prism.js。所有案例資料以 JavaScript 陣列與 JSON 儲存，收藏與偏好設定儲存於 localStorage。
